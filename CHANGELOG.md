# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-02-04

### Added

- Privacy-first RAG platform positioning emphasizing local-first architecture
- Support for multiple document types beyond code (contracts, PDFs, documentation)
- VS Code extension for inline knowledge search
- MCP server for AI agent integration
- Command-line interface for indexing and search
- Real-time file watching with incremental indexing
- Semantic search across multiple document collections
- RAG-powered chat interface
- Support for multiple AI providers (OpenAI, Groq, Gemini, Anthropic)
- PostgreSQL + pgvector integration for semantic search
- Tree-sitter integration for intelligent code parsing
- Flexible document classification system
- Knowledge graph visualization
- Comprehensive documentation and deployment guides

### Changed

- Restructured as independent services in monorepo (removed shared package)
- Updated architecture for multi-domain applicability
- Improved build configuration and optimization

### Fixed

- Build and linting issues
- Documentation consistency

## [1.0.0] - 2025-01-XX

### Added

- Initial release
- Core RAG functionality for codebases
- Basic search and chat features
- Web UI and CLI
